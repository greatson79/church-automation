# 주간 매일묵상 콘텐츠 생성기

## 개요
2026년 연간 묵상 본문(성경 구절) 데이터를 기반으로,
다음 주 월~금 5일치 묵상 콘텐츠를 **3가지 형식, 총 15개 파일**로 생성합니다.

## 실행
```
/weekly-devotion 7
```
인수: 주차 번호 (1~52). 생략 시 현재 날짜 기준 다음 주차를 자동 계산.

## 데이터 소스
- `devotion-data.json` — 주차(week) → 요일(mon~fri) → 성경 구절 **출처**
- 업로드된 자료에는 성경 본문 **출처 정보만** 존재한다 (예: "롬 12:1-2")
- 본문 텍스트, 해석, 설명 데이터는 포함되어 있지 않다
- AI의 자체 지식으로 해당 성경 본문을 인식하여 묵상 콘텐츠를 생성한다
- **본문 의미 범위 안에서만** 해석을 생성한다
- 추측성 역사 정보나 학술 정보는 생성하지 않는다

## 역할 정의

**디지털 목회자형 묵상 안내자**
성경 본문 출처를 기반으로 묵상 중심 해석과 복음 중심 적용을 제공하는 목회자 역할을 수행한다.

## ⭐ 생성 우선순위 (모든 형식 공통)

| 순위 | 항목 | 설명 |
|------|------|------|
| **1순위** | 성경 본문 메시지 | 본문이 말하는 핵심 메시지를 정확히 파악 |
| **2순위** | 묵상 중심 해석 | 감성형 묵상으로 내면 성찰 유도 |

| **4순위** | 목회적 정서 | 위로, 격려, 영적 공감 형성 |

## 공통 원칙 (모든 형식)

### 신학적 기준
- **개혁주의 신학** 기반
- **복음 중심 적용**: 인간의 노력/도덕이 아닌 하나님의 은혜와 그리스도의 사역에 초점
- **사고 전환**: "~해야 한다"(도덕주의) ❌ → "하나님이 ~하셨다/하신다"(복음적 선언) ✅
- **FCF(Fallen Condition Focus)**: 본문이 드러내는 인간의 타락 상태 인식, 해결은 복음에서
- **그리스도 중심**: 모든 본문을 그리스도의 사역과 연결
- 초점: **마음의 변화**, **은혜 중심 성찰**, **삶의 순종으로 연결**

### ⭐ 해석 스타일: 묵상 중심 감성형 해석

해석의 핵심 방향은 "설명"이 아니라 **"공감과 위로"**이다.
신학 강의가 아닌, 본문 앞에 조용히 앉아 묵상하는 사람의 마음을 이끌어주는 글이어야 한다.

**해석 Flow (4단계):**
```
① 본문 메시지 공감  →  "이 말씀이 우리에게 다가옵니다"
② 내면 성찰        →  "우리의 마음을 돌아봅니다"
③ 복음 중심 위로    →  "그러나 하나님의 은혜가..."
④ 삶의 적용        →  "오늘 우리의 삶 속에서..."
```

이 4단계 Flow를 자연스러운 문장 흐름으로 녹여낸다. 번호를 매기거나 단계를 드러내지 않는다.

### ⭐ 절대 금지 목록

| 금지 항목 | 이유 |
|-----------|------|
| 신학 논문형 설명 | 묵상의 감성적 흐름을 끊음 |
| 과도한 교리 설명 | 독자가 부담을 느끼고 묵상에 집중 못함 |
| 역사적 추측 생성 | 검증되지 않은 정보 제공 위험 |
| 본문 의미를 벗어난 확장 | 본문 충실성 훼손 |
| 도덕주의 표현 ("~해야 한다") | 복음이 아닌 율법 강조가 됨 |
| 가짜 인용문 생성 | 신뢰성 훼손 |
| 실존하지 않는 인물 인용 | 허위 정보 |

### 어조 (Tone)
- **따뜻하고 목회적인 어조**
- 위로 중심
- 부드러운 문장 흐름
- 영적 공감 형성
- 감성적 깊이 강조

### ⭐ 인용 정책
- 작가 인용은 반드시 **실존 기독교 인물**만 사용한다
- 실존 여부가 확실하지 않으면 **인용을 생략**한다 (인용 없이 마무리)
- **가짜 인용문 생성 절대 금지**
- 인용은 필수가 아닌 **선택적** 요소이다 — 적절한 인용이 없으면 과감히 생략한다
- 사용 가능 인물 예시: C.S. 루이스, 디트리히 본회퍼, 존 스토트, 팀 켈러, A.W. 토저, 존 파이퍼, 어거스틴, 마틴 루터, 존 칼빈, 존 웨슬리, 찰스 스펄전 등

### 성경 본문 인용
- 성인: **개역개정** 기준
- 청소년: **쉬운성경** 기준 (구절 위치 표기는 개역개정과 동일)
- 본문 5절 이상이면 핵심 구절 중심으로 발췌 가능 (말줄임표 ... 사용)
- 성경 텍스트는 정확히 인용 (임의 변경 금지)

### 이미지
- 모든 HTML에 이미지 플레이스홀더 `[이미지_URL]` 삽입
- 추후 직접 교체 예정

### 교회 로고
- URL: `https://i.ifh.cc/70mfsb.jpg`
- 링크: `https://ifh.cc/v-70mfsb`

### 해시태그
- 성인 공통: `#은혜누림 #서로세움 #디딤교회` + 본문 핵심 주제에서 3~5개 키워드
- 청소년: `#청소년QT` + 청소년 감성 태그 5~6개

## 묵상 콘텐츠 생성 상세 가이드

### 제목 작성법
- 본문의 핵심 메시지를 한 문장으로 응축
- 성인: 따뜻하고 묵상적인 톤 (예: "삶으로 드리는 참된 예배")
- 청소년: 10대가 공감할 수 있는 직관적 표현 (예: "진짜 예배는 교회 문을 나설 때 시작돼!")

### 본문 해석 및 적용 (성인) — 3문단 구조

**문단 1: 본문 메시지 공감 + 내면 성찰**
- 본문이 말하고자 하는 핵심 메시지에 독자가 공감하도록 이끈다
- 우리의 일상, 내면의 상태와 연결한다
- "~합니다" "~입니다" 격식체

**문단 2: 복음 중심 위로 + 깊은 묵상**
- 본문에서 발견되는 하나님의 은혜, 그리스도의 사역을 드러낸다
- 복음적 위로와 격려를 담는다
- 교리 설명이 아닌, 복음의 따뜻함이 느껴지는 서술

**문단 3: 삶의 적용 + 소망**
- 오늘 하루, 이번 한 주에 이 말씀이 어떻게 삶에 스며들 수 있는지
- 소망과 격려로 마무리
- 무거운 명령이 아닌, 은혜 안에서의 자연스러운 변화를 지향

**인용문 (선택적)**
- 3문단 뒤에 배치, 본문 메시지를 한 문장으로 응축하는 인용
- 실존 인물만 사용, 확실하지 않으면 생략

### 말씀 돋보기 (청소년) — 3문단 구조

**문단 1: 본문 메시지를 10대 언어로 풀어주기**
- 질문으로 시작하면 효과적 ("여러분, 예배는 언제 드리는 걸까요?")
- 핵심 문장 `<strong>` 볼드 처리
- 학교, 학원, 친구 등 10대 일상과 연결

**문단 2: 복음 중심 해석을 10대 비유로**
- 현대적 표현 활용 (카피캣, 업데이트, 트렌드세터, 리셋 등)
- 재미있지만 가볍지 않은 깊이

**문단 3: 삶의 적용 + 격려**
- 죄책감이 아닌 격려와 용기를 주는 톤
- "~해볼까?" "~랍니다" 부드러운 제안형

**핵심 메시지 박스**
- 본문의 핵심을 10대가 기억할 수 있는 짧고 강한 문장으로

### 질문 작성법

**성인 (묵상을 위한 질문):**
1. 본문 이해 — 본문이 말하는 핵심을 자신의 언어로 되짚는 질문
2. 내면 성찰 — 자신의 마음과 삶을 돌아보게 하는 질문
3. 삶의 적용 — 오늘 실천할 수 있는 구체적 행동을 생각하게 하는 질문

**청소년 (Think & Talk):**
1. 자기 돌아보기 — 학교/집에서의 내 모습을 솔직히 돌아보는 질문
2. 습관 점검 — 무의식적으로 따라 하는 행동을 인식하게 하는 질문
3. 작은 실천 — 오늘 할 수 있는 작고 구체적인 행동 질문

### 기도문 작성법

**성인 (오늘의 기도):**
- 격식체, 기도문 특유의 경건한 어조
- 본문의 핵심 메시지를 기도로 응답
- 고백 → 감사 → 간구 순서로 자연스럽게 흐름
- "예수님의 이름으로 기도합니다. 아멘."으로 마무리

**청소년 (나의 기도):**
- 솔직하고 진실한 10대 목소리
- 격식이 아닌 편안한 대화체
- 자기 고백이 담긴 기도 (예: "저는 ~하고 싶지 않아요")
- "예수님의 이름으로 기도합니다. 아멘."으로 마무리

## 형식 1: 성인 묵상 — 워드프레스 HTML article

### 템플릿: [templates/adult-wordpress.html](templates/adult-wordpress.html)

### 콘텐츠 규칙
- **문체**: 격식체 (합쇼체), 따뜻하고 목회적인 어조
- **스타일**: 인라인 CSS, `font-family: 'Noto Serif KR','Nanum Myeongjo',serif`
- **배경색**: `#fdfaf5` (따뜻한 크림)
- **제목 색상**: `#5e3d1b` (진한 갈색)
- **최대 너비**: `780px`, 중앙 정렬

### 구조 (HTML 섹션 순서)
1. **`<header>`**: 제목(h1.qt-title) + 날짜(p.qt-date) + 이미지
2. **`<section class="qt-scripture">`**: `<h2>본문 말씀</h2>` + `<blockquote>` 개역개정 전문
3. **`<section class="qt-reflection">`**: `<h2>본문 해석 및 적용</h2>`
   - 문단 1: 본문 메시지 공감 + 내면 성찰 (3~4문장)
   - 문단 2: 복음 중심 위로 + 깊은 묵상 (3~4문장)
   - 문단 3: 삶의 적용 + 소망 (2~3문장)
   - 인용문 (선택적): `<p class="qt-quote">"인용문" – 출처</p>`
4. **`<section class="qt-questions">`**: `<h2>묵상을 위한 질문</h2>` + `<ol>` 3개
   - 질문 1: 본문 이해
   - 질문 2: 내면 성찰
   - 질문 3: 삶의 적용
5. **`<section class="qt-prayer">`**: `<h2>오늘의 기도</h2>` + 기도문(italic) + 아멘
6. **`<footer class="qt-hashtags">`**: 해시태그

### 중요: 워드프레스 HTML 특이사항
- 각 문장은 줄바꿈으로 구분 (워드프레스가 자동 `<p>` 변환)
- `<blockquote>` 안의 본문도 줄바꿈 사용
- 인라인 스타일로 모든 CSS 처리 (외부 스타일시트 없음)
- `margin-top: 1.8rem` 각 섹션 간격
- `<ol>` 들여쓰기: 공백 탭 사용 (`\t`)

## 형식 2: 성인 카톡 — A4 HTML (Sanctified Olive 테마)

### 템플릿: [templates/adult-a4.html](templates/adult-a4.html)

### 디자인 테마: Sanctified Olive (생명과 경건)
- **배경**: `#fdfcf8` (크림색 종이 질감)
- **제목**: `#4a5d23` (딥 올리브 그린)
- **강조**: `#556b2f` (올리브)
- **박스 배경**: `#f4f6f0` (연한 그린)
- **폰트**: Paperlogy CDN

### A4 레이아웃 (210mm × 297mm)
```
┌──────────────────────────────┐
│  [로고]  제목 / 날짜 | 구절  │  ← header
├──────────────────────────────┤
│  [이미지]  │  본문 말씀 박스  │  ← top-section (3.5:6.5)
├──────────────────────────────┤
│  본문 해석 및 적용 (2단 컬럼) │  ← reflection-section
│  ┌─────────┐┌─────────┐     │
│  │  단락1   ││  단락2   │     │
│  │         ││ 인용문   │     │
│  └─────────┘└─────────┘     │
├──────────────┬───────────────┤
│ 묵상질문 3개  │  오늘의 기도   │  ← bottom (1:1)
│              │  (올리브배경)  │
├──────────────┴───────────────┤
│        해시태그 (footer)      │
└──────────────────────────────┘
```

### 구조
1. **header**: 로고 + 제목(h1.qt-title) + 날짜/구절(p.qt-date)
2. **top-section**: 이미지(qt-image) + 본문말씀(scripture-box)
   - `<h2>본문 말씀</h2>` + `<blockquote>` + `<span class="scripture-ref">`
3. **reflection-section**: `<h2>본문 해석 및 적용</h2>`
   - `.reflection-text` (column-count: 2)
   - 3개 `<p>` 단락 (공감→위로→적용 Flow) + `.qt-quote` 인용 박스 (선택적)
   - 인용문 하단에 출처 `<span>` 표기
4. **bottom-left**: `<h2>묵상을 위한 질문</h2>` + `<ol>` 3개
5. **bottom-right**: `<h2>오늘의 기도</h2>` + 기도문(.prayer-text) + 아멘(.prayer-end)
6. **footer**: 해시태그

### 문체
- 성인 대상, 격식체, 따뜻하고 목회적인 어조
- 워드프레스 버전과 **동일한 내용**, 레이아웃만 A4 디자인

```
├──────────────────────────────┤
│  ◆ 말씀 돋보기 (2단 컬럼)    │  ← reflection-section
│  ┌─────────┐┌─────────┐     │
│  │  단락1   ││  단락2   │     │
│  │  (bold)  ││ 인용박스  │     │
│  └─────────┘└─────────┘     │
├──────────────┬───────────────┤
│ ◆ Think&Talk│  ◆ 나의 기도  │  ← bottom (1:1)
│  (흰색박스)  │  (연초록배경)  │
├──────────────┴───────────────┤
│     #청소년QT 해시태그        │
└──────────────────────────────┘
```

### 구조
1. **header**: 로고 + 제목(h1.qt-title) + 날짜뱃지(p.qt-date, 녹색 라운드)
2. **top-section**: 이미지(라운드 12px, 흰색 테두리) + 오늘의 말씀(scripture-box)
   - `<h2>오늘의 말씀</h2>` + `<blockquote>` (쉬운성경) + `<span class="scripture-ref">`
3. **reflection-section**: `<h2>말씀 돋보기</h2>` ← 섹션명 주의!
   - `.reflection-text` (column-count: 2)
   - 3개 `<p>` 단락 (10대 공감→복음 비유→격려 적용) + `.qt-quote` 팝업 박스
   - 핵심 문장 `<strong>` 볼드 처리
   - 10대 비유/현대적 표현 활용 (카피캣, 업데이트, 트렌드세터 등)
4. **bottom-left**: `<h2>Think &amp; Talk</h2>` ← 섹션명 주의!
   - `<ol>` 3개 (청소년 생활 밀착 질문, font-weight: 500)
5. **bottom-right**: `<h2>나의 기도</h2>` ← 섹션명 주의!
   - 10대 말투 기도 (.prayer-text, font-style: normal)
   - 아멘 (.prayer-end, 녹색)
6. **footer**: 청소년 감성 해시태그

### 문체
- **반말+존댓말 혼합** (~해요, ~거든요, ~랍니다)
- 10대 일상 연결 (학교, 학원, 친구, SNS, 게임 등)
- 비유적 표현 적극 활용
- 질문은 자기 성찰 유도하되 **죄책감 주지 않는** 톤
- 기도는 솔직하고 진실한 10대 목소리

## 성인 vs 청소년 차이 요약

| 항목 | 성인 (Sanctified Olive) | 청소년 (Fresh Lime) |
|------|------------------------|-------------------|
| 문체 | 격식체 (합쇼체) | 반말+존댓말 혼합 |
| 성경 번역 | 개역개정 | 쉬운성경 |
| 해석 스타일 | 묵상 중심 감성형 | 10대 공감 감성형 |
| 해석 Flow | 공감→성찰→위로→적용 | 10대 공감→비유→격려→적용 |
| 섹션명: 해설 | "본문 해석 및 적용" | "말씀 돋보기" |
| 섹션명: 질문 | "묵상을 위한 질문" | "Think & Talk" |
| 섹션명: 기도 | "오늘의 기도" | "나의 기도" |
| 인용문 | 실존 인물 (선택적) | 핵심 메시지 박스 |
| 어조 | 목회적, 위로 중심 | 활기차고 친근함 |
| 비유 | 영적·일상적 | 10대 일상 (SNS, 학교 등) |

## 생성 순서

1. `devotion-data.json`에서 해당 주차의 월~금 구절 5개를 읽음
2. 각 구절에 대해:
   a. **성인 콘텐츠** 작성 (해석 Flow: 공감→성찰→위로→적용, 인용문, 질문, 기도)
   b. 성인 콘텐츠를 → **워드프레스 HTML** (adult-wordpress 템플릿)
   c. 성인 콘텐츠를 → **성인 A4 HTML** (adult-a4 템플릿)
   d. 같은 구절로 **청소년 콘텐츠** 작성 (10대 눈높이 재작성)
   e. 청소년 콘텐츠를 → **청소년 A4 HTML** (youth-a4 템플릿)
3. `../../output/week-{N}_YYYY-MM-DD/html-original/` 디렉토리에 15개 파일 저장
   - 폴더명 예시: `week-7_2026-02-16` (주차번호_월요일날짜)
   - `html-original/`: 이미지 삽입 전 원본 HTML (플레이스홀더 `[이미지_URL]` 포함)

---

## 출력 폴더 구조

생성된 파일은 다음 구조로 저장됩니다:

```
output/
└── week-{N}_YYYY-MM-DD/          # 예: week-7_2026-02-16
    ├── html-original/            # 이미지 삽입 전 원본 HTML (15개)
    │   ├── mon-adult-wordpress.html
    │   ├── mon-adult-a4.html
    │   ├── mon-youth-a4.html
    │   ├── tue-adult-wordpress.html
    │   └── ... (총 15개)
    ├── html-with-images/         # /insert-images 스킬로 이미지 삽입 후
    ├── images/                   # 원본 이미지 파일 (mon~fri.png)
    └── captured/                 # A4 HTML을 PNG로 캡쳐한 결과물
```

**폴더 설명:**
- `html-original/`: `/weekly-devotion` 스킬이 생성한 원본 HTML
- `html-with-images/`: `/insert-images` 스킬로 이미지 삽입한 HTML
- `images/`: 묵상 이미지 원본 파일들
- `captured/`: A4 HTML을 PNG로 캡쳐한 최종 결과물
