사용자가 요청한 기도제목 문서를 생성하라.

## 입력 파싱

`$ARGUMENTS`에서 **월**과 **주차**를 추출한다.

예시:
- "3월 2주차" → 월=3, 주차=2
- "8월 첫째 주" → 월=8, 주차=1
- "12월 넷째주" → 월=12, 주차=4

주차가 없으면 사용자에게 확인 요청한다.

## 실행

```bash
python3 .claude/skills/prayer-doc/scripts/run_pipeline.py \
  data/*.csv <월_숫자> <주차_숫자> \
  --base-dir .
```
