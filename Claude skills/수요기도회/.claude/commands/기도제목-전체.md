사용자가 요청한 월의 **모든 주차** 기도제목 문서를 일괄 생성한다.

## 입력 파싱

`$ARGUMENTS`에서 **월**을 추출한다.
예시: "8월" → 월=8

## 실행

1~5주차까지 순서대로 실행한다. 데이터가 비어있는 주차는 건너뛴다.

```bash
for week in 1 2 3 4 5; do
  python3 .claude/skills/prayer-doc/scripts/run_pipeline.py \
    data/*.csv <월_숫자> $week \
    --base-dir .
done
```

## 결과 안내

생성된 모든 파일 목록을 안내:
```bash
find output/<월>월 -type f -name "*.png" | sort
```
